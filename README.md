# RID181650_Desafio02
📱 Blog Responsivo
Projeto desenvolvido como desafio do módulo de Responsividade com CSS da DNC Interactive. O objetivo foi transformar um blog estático em uma experiência completamente responsiva, adaptada para smartphones, tablets e monitores ultrawides.

🎯 Sobre o projeto
O desafio consistiu em analisar um layout no Figma e recriá-lo do zero, garantindo que todos os elementos se adaptem corretamente a qualquer tamanho de tela — sem perder identidade visual ou funcionalidade.

✨ Funcionalidades

Layout fluido que se adapta a mobile, tablet e desktop
Navegação com menu hamburguer para telas pequenas
Grid de cards com 3 → 2 → 1 coluna conforme o breakpoint
Tipografia fluida com clamp() para escala suave entre dispositivos
Seções de features com Flexbox que empilham no mobile
Tags e filtros de tópicos com flex-wrap
Animações de entrada com Intersection Observer


🛠️ Tecnologias

HTML5 semântico
CSS3 — Flexbox, Grid, Media Queries, Custom Properties
JavaScript vanilla (menu mobile e animações de scroll)
Fonte Montserrat via Google Fonts
